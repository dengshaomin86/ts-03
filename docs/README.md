# 配置

## 安装依赖

- 1. npm init -y
- 2. npm i -D webpack webpack-cli typescript ts-loader
- 3. html 生成插件 npm i -D html-webpack-plugin
- 4. webpack 开发服务器 npm i -D webpack-dev-server
- 4. 清除 dist 打包目录 npm i -D clean-webpack-plugin
